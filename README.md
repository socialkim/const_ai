# 현장의 언어가 된 AI — 건설사 AI 실습 키트 (범용)

ChatGPT · MCP를 활용한 **건설 실무 AI 실습 자료**입니다.
특정 회사에 종속되지 않은 범용 버전으로, 어느 건설사에서나 그대로 사용·배포할 수 있습니다.

제작: **김덕진 · IT커뮤니케이션연구소(ITCL)**

## ▶️ 사용법

1. `index.html` 을 브라우저에서 엽니다. (GitHub Pages로 올리면 모바일에서 바로 접속)
2. **‘④ 파일 실습’** 탭에서 위 데이터 파일을 ChatGPT에 올린 뒤, 카드의 **복사** 버튼으로 프롬프트를 붙여넣습니다.
3. **MCP 실습**(법령·건축HUB·KOSIS)은 **‘① 준비 · 연결’** 탭 안내대로 ChatGPT 개발자 모드에 서버를 연결한 뒤 사용합니다.
4. 프롬프트 속 **`[우리 회사]` · `[우리 단지]` · `[지역]` · `[주소]`** 등 대괄호 부분만 각자 상황으로 바꾸면 됩니다.

## 🌐 GitHub Pages 배포 (선택)

저장소에 이 폴더를 올린 뒤 **Settings → Pages → Branch: main / root** 로 설정하면,
`index.html` 이 저장소 URL에서 바로 서비스됩니다. 그 주소로 QR을 만들어 강의에 활용하세요.
